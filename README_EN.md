## ⚡️ Project Overview

**Latexia** is a **full-featured online LaTeX learning and practice platform** for students, researchers, and technical writers. Built with a Monorepo architecture using **Next.js 14 + Hono + Drizzle ORM**, Latexia provides an all-in-one, gamified, and immersive learning experience to address the steep learning curve of academic typesetting.

The project features **systematic tutorials**, **comprehensive exercise banks**, **timed formula training**, a **LaTeX Draftpad** (visual editor), **intelligent error review (Ebbinghaus curve)**, **leaderboards**, and **learning visualization**. It even includes a built-in **immersive music player** to help users stay focused and enjoy the learning process.

## ✨ Features

<div align="center">

| Core Feature               | Description                                                                                                            |
| :------------------------- | :--------------------------------------------------------------------------------------------------------------------- |
| **Problem Training**       | Extensive LaTeX problem bank categorized by difficulty and domain, with instant validation and detailed explanations.  |
| **Tutorial Modules**       | Systematically organized chapters from basics to complex math. Supporting Markdown + KaTeX immersive reading with TOC. |
| **Formula Training**       | Timed training mode designed for efficient memorization of **3000+** formulas across math and physics.                 |
| **Bookmarks**              | Save difficult problems or frequent formulas with custom folder management.                                            |
| **Error Review**           | Automatically archive wrong answers. Intelligent review plans based on the **Ebbinghaus Forgetting Curve**.            |
| **LaTeX Draftpad**         | Visual/shortcut formula editor similar to MathType with real-time preview and code extraction.                         |
| **Leaderboard**            | Build motivation with multi-dimensional rankings (Total, Accuracy, Speed).                                             |
| **Learning Visualization** | Visualize your growth with activity heatmaps, knowledge radar charts, and progress tracks.                             |
| **Symbol Search**          | Comprehensive symbol dictionary with keywords search, one-click copy, and usage examples.                              |
| **Resource Hub**           | Curated list of LaTeX editors, official docs, and productivity tools.                                                  |
| **Music Player**           | Built-in Lofi/Ambient player with floating lyrics to make learning more enjoyable.                                     |
| **i18n Support**           | Seamless switching between Chinese and English to support developers worldwide.                                        |
| **Theme Switching**        | Premium Light/Dark mode support with smooth transitions.                                                               |
| **OAuth Support**          | Fast login via GitHub, Google, etc.                                                                                    |

### Overview

- **Frontend**: Next.js 14 (App Router), React 18, Tailwind CSS, Radix UI / `@latexia/ui`.
- **Backend**: Hono 4, Drizzle ORM, PostgreSQL, Redis.
- **Monorepo**: pnpm workspace + Turbo.

## 🚀 Deployment Guide

This project is built with a separate frontend and backend, fully dependent on the Node.js environment.

### 1. Prerequisites

#### 1.1 Install NVM & Node.js

We recommend using **NVM** (Node Version Manager) to manage Node versions.

- **Windows**: Download `nvm-setup.exe` from [nvm-windows Releases](https://github.com/coreybutler/nvm-windows/releases).
- **macOS/Linux**: Run the installation script:
  `bash
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
    `
  Restart your terminal and install Node.js **20.x** LTS:

```bash
nvm install 20
nvm use 20
node -v  # Should output v20.x
```

#### 1.2 Install pnpm

This project strictly requires `pnpm`:

```bash
npm install -g pnpm
pnpm -v  # Should output 9.x
```

#### 1.3 Install PostgreSQL

- **Windows/macOS**: Download from [PostgreSQL Official Website](https://www.postgresql.org/download/).
- **Docker (Recommended)**:
  ```bash
  docker run --name latexia-pg -e POSTGRES_PASSWORD=your_password -p 5432:5432 -d postgres
  ```

#### 1.4 Install Redis

- **macOS**: `brew install redis && brew services start redis`
- **Docker**:
  ```bash
  docker run --name latexia-redis -p 6379:6379 -d redis
  ```

---

### 2. Database Initialization

1.  **Create Database**: Create a database named `latexia` using PgAdmin, DBeaver, or `psql`.
    ```sql
    CREATE DATABASE latexia;
    ```
2.  **Import Data**: Import the `latexia_init.sql` script into the database.
    ```bash
    psql -U postgres -d latexia -f ./path/to/latexia_init.sql
    ```

---

### 3. Configuration & Launch

#### 3.1 Install Dependencies

```bash
git clone https://github.com/Auroral0810/LaTexia.git
cd Latexia
pnpm install
```

#### 3.2 Environment Variables (`.env`)

Configure the backend environment variables:

```bash
cd apps/api
cp .env.example .env
```

Edit `apps/api/.env` and provide your database and Redis URLs.
Example:

```env
DATABASE_URL=postgresql://latexia:password@localhost:5432/latexia
REDIS_URL=redis://localhost:6379
JWT_ACCESS_SECRET=your_secret
JWT_REFRESH_SECRET=your_secret
```

#### 3.3 Start Services

From the **root directory**, run Turbo to start both Frontend and Backend:

```bash
# In the root directory
pnpm run dev
```

**Verification**:

1.  Visit **http://localhost:3001/doc/ui** for the Swagger UI.
2.  Visit **http://localhost:3000** for the Latexia Frontend! 🎉
